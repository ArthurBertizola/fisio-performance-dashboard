Este é um Web App de Business Intelligence (BI) customizado, desenvolvido para transformar dados brutos de avaliações físicas em decisões clínicas estratégicas. O sistema foi projetado para monitorar a saúde musculoesquelética de atletas, identificar assimetrias críticas e prevenir lesões.

## 🧠 Visão de Negócio (Economia + ADS)
Este projeto reflete minha capacidade de unir a análise estatística (baseada na minha formação em Economia) com a engenharia de software (ADS). O objetivo central é a **preservação do ativo (atleta)**, minimizando o tempo fora de campo e otimizando a performance através de dados.

## 🛠️ Funcionalidades Técnicas
- **ETL Automatizado**: Processamento e limpeza de planilhas Excel utilizando `Pandas`.
- **Análise de Benchmarking**: Cálculo dinâmico de médias por posição (Zagueiras, Atacantes, etc.), permitindo comparar a performance individual com a referência do elenco.
- **Detecção de Assimetrias**: Algoritmo que calcula automaticamente disparidades > 15% entre membros dominantes e não-dominantes.
- **Visualização de Dados**: Gráficos interativos em `Plotly` com indicadores qualitativos (Verde/Amarelo) para testes de mobilidade (Thomas e Agachamento).
- **Histórico de Evolução**: Monitoramento temporal das avaliações para acompanhar a recuperação ou ganho de performance.

## 💻 Tecnologias Utilizadas
- **Linguagem**: Python
- **Manipulação de Dados**: Pandas, NumPy
- **Visualização**: Plotly Express/Graph Objects
